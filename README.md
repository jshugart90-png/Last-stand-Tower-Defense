# Last Stand Tower Defense

Mobile-first tower defense game with progression systems, daily challenge support, and a FastAPI backend that supports Supabase.

## Project Structure

- `frontend/`: Expo React Native client
- `backend/`: FastAPI API service
- `scripts/`: release and automation scripts

## Quick Start

```bash
npm install
npm run start
```

## Release and Deploy

- Run release checks: `npm run qc:release`
- Run lints: `npm run lint`
- Backend deploy guide: `DEPLOY_BACKEND.md`
- Full launch checklist: `LAUNCH_CHECKLIST.md`
- Current release handoff notes: `RELEASE_HANDOFF.md`
